# Compliance IEGM Dashboard

Dashboard do IEGM da Prefeitura de Betim.

## Descricao

Dashboard desenvolvido em Next.js para acompanhamento do Indice de Efetividade da Gestao Municipal (IEGM) do TCE-MG.

## Tecnologias

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Recharts

## Configuracao

1. Clone o repositorio
2. Execute `npm install`
3. Configure as variaveis de ambiente
4. Execute `npm run dev`

## Estrutura

```
app/           # Rotas e paginas
components/    # Componentes React
lib/           # Utilitarios e funcoes
public/        # Arquivos estaticos
```
